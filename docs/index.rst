Welcome to MetaCarvel's documentation!
===================================

|

MetaCarvel is an improved version of a metagenomic scaffolder Bambus2. MetaCarvel works with standard file formats and easier to use compared to Bambus2. 
We have made significant changes in the repeat detection module thus making it order of magnitude faster than Bambus2.
